# Code Tours

This directory contains VS Code code tours that provide interactive walkthroughs of the repository.

## Available Tours

- **[Getting Started](getting-started.tour)**: A guided tour for new contributors to understand the repository structure and key files.

## How to use

1.  Install the [CodeTour extension](https://marketplace.visualstudio.com/items?itemName=vsls-contrib.codetour) in VS Code.
2.  Open the CodeTour panel in the Explorer view.
3.  Select a tour to start.
